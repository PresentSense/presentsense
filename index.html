Project Overview

Our senior project focuses on developing a feedback system that enhances presentation skills. The system will allow users to present and receive real-time feedback from peers, professionals, and automated systems powered by machine learning algorithms.

The core functionality includes video recording of presentations, an analysis module that identifies areas of improvement (e.g., voice modulation, pacing, eye contact), and a dashboard for viewing feedback in an easy-to-understand manner.

This project aims to help presenters of all levels improve by offering insightful feedback and identifying areas that need work, making the feedback process more accessible and efficient.
